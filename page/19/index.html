检测MYSQL不同步发邮件通知的脚本

脚本代码如下：

################################

#!/bin/bash 

# Info   : check mysql slave                                                                                              

# Author : dingtm                                                                                                         

# CTime  : 2011.03.21                                                                                                     

# This script run by root                                                                                                 



DBDir=/elain/apps/mysql/bin/                                                                                              

DBSock=/elain/data/mysql/3306/mysql.sock                                                                                  

Log=/tmp/mysql_slave.log                                                                                                  

DBUser=root                                                                                                               

DBPasswd=elain                                                                                                       

MailList=elain@elain.org,elain2012@hotmail.com                                                                               



Seconds_Behind_Master=$(${DBDir}mysql -u${DBUser} -p${DBPasswd} -S ${DBSock} -e &quot;show slave status\G;&quot; | awk -F&#039;:&#039; &#039;/Seconds_Behind_Master/{print $2}&#039;)                                                                                             



if [ ${Seconds_Behind_Master} != &quot;NULL&quot; ];                                                                                

    then                                                                                                                  

         echo &quot;slave is ok!&quot;

Ubuntu运行wine QQ(比较完美)

偶然的机会，发现这个个WINE QQ，个人感觉效果不错，大家值得一试，各个功能测试了一下都比较完美，是迄今为至比较完美的WINE QQ啦

qq2010-for-wine下载：

http://www.longene.org/download/qq-for-wine_20110510.deb

MD5:fd71ce314ba480b6b2144bf16af5b6d1

SHA1:59677588625fe72df1332e61ee08b72f80668c59

安装说明：

1.安装: sudo dpkg -i 软件名.deb

卸载: sudo dpkg -r qq-for-wine.deb 或 sudo dpkg -P qq-for-wine.deb

2.安装后qq在桌面上方的Applications/Internet/Tencent QQ，点击运行即可，也可以自己拖到桌面双击运行

或者 到/usr/local/wineapp/qq目录下运行qq.sh[感谢Ivenvd网友的脚本]

3.安装后在你的 $HOME下会生成.qq目录，这是qq的安装目录

4.安装后第一次运行qq登录的时候可能有点慢这是正常的，qq要生成自己的一些用户信息

每次运行会出现安装更新提示，选择 安装，然后在下一个窗口点关闭，等一下就会出现登陆窗口
