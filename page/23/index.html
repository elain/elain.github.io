Ubuntu下使用windows字体

1、将需要安装的字体复制到某个目录，我是将要安装的新宋体（simsun.ttc）和微软雅黑（msyh.ttf）复制到/bak/font目录。

2、打开终端，依次输入以下命令

sudo mkdir /usr/share/fonts/msfonts //创建一个名为msfonts的目录
sudo cp /bak/font/ /usr/share/fonts/msfonts //将/bak/font目录下的字体复制到msfonts目录中
cd /usr/share/fonts/msfonts //进入创建的字体目录
sudo chmod 744
sudo mkfontscale
sudo mkfontdir

Pptpd安装部署文档

简述：

PPTP：点对点隧道协议
(PPTP: Point to Point Tunneling Protocol）
点对点隧道协议（PPTP）是一种支持多协议虚拟专用网络的网络技术,它工作在第二层。通过该协议，远程用户能够通过 Microsoft Windows NT 工作站、Windows xp 、 Windows 2000 和windows2003操作系统以及其它装有点对点协议的系统安全访问公司网络，并能拨号连入本地 ISP，通过 Internet 安全链接到公司网络。

检查 MPPE
modprobe ppp-compress-18 && echo ‘ok! MPPE was found !’

[root@elain tools]# modprobe ppp-compress-18 && echo ‘ok! MPPE was found !’
ok! MPPE was found ! //说明系统存在MPPE
