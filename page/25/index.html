CentOs5.4设置DVD为本地yum源

进入 /etc/yum.repo.d/ 文件夹

vi Centos-Media.repo

修改文件中的dvd的位置，并打开本地源，如下：

CentOS-Media.repo

#

This repo is used to mount the default locations for a CDROM / DVD on

CentOS-5. You can use this repo and yum to install items directly off the

DVD ISO that we release.

#

To use this repo, put in your DVD and use it with the other repos too:

yum –enablerepo=c5-media [command]

网络传输边传输边压缩方法

A机上传送16G文件夹到B机上打包的方法

先在B机上运行：

nc -l 1985 > bak.tar.gz

再在A机上运行：

tar cz . | nc 192.168.9.101 1985

日志归总脚本部署思路

服务端

wget http://rsync.samba.org/ftp/rsync/src/rsync-3.0.7.tar.gz
rpm -e –nodeps rpm -qa |grep rsync
tar zxvf rsync-3.0.7.tar.gz
cd rsync-3.0.7
./configure –prefix=/elain/apps/rsync
make && make install
cd ..
ln -s /elain/apps/rsync/bin/rsync /bin/rsync
