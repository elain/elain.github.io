2. 修改my.ini(my.cnf) ，在 [mysqld] 后面加入一行“ft_min_word_len=1”，然后重启Mysql，再登录网站后台（模块管理->全站搜索）重建全文索引。
3. 记录慢查询sql语句，修改my.ini(my.cnf)，添加如下代码：
4. #log-slow-queries
5. long_query_time = 1 #是指执行超过多久的 sql 会被 log 下来
6. log-slow-queries = E:/wamp/logs/slow.log #设置把日志写在那里，可以为空，系统会给一个缺省的文件
7. #log-slow-queries = /var/youpath/slow.log linux下 host_name-slow.log
8. log-queries-not-using-indexes
